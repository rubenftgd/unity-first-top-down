# unity-first-top-down

2D game created in Unity from youtube video "TOP DOWN MOVEMENT in Unity!" by Brackeys.

Link to the Video:

https://www.youtube.com/watch?v=whzomFgjT50

Link To his channel:

https://www.youtube.com/@Brackeys
